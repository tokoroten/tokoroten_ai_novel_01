## フォルダ構成
- `ideas/`  - アイデアやプロットのメモ、制作方針など
  - `ideas/character/` - キャラクターの設定、SD用のプロンプトなど
  - `ideas/plot/` - プロット、あらすじ、ストーリーの流れなど
- `novel/` - ノベルの原稿
- `images/` - ノベルに使用する画像
- `scripts/` - 挿絵生成に使ったスクリプト
  - ローカルで起動している automatic1111-webui を叩いて、挿絵を生成する
